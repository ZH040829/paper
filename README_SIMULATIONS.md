## 🚀 运行方法

### 环境依赖
```bash
pip install numpy matplotlib torch torchvision
```

### 1. 时间场模拟
```bash
python time_field_simulation.py
```

**输出**：
- `time_field_simulation.png`：熵密度分布与最终时间场分布
- `time_field_evolution.gif`：时间场演化动画

**验证预言**：
在低熵区域中心出现时间场负值（时间-相态），符合论文4.2节时间三态模型。

### 2. 神经网络熵减模拟
```bash
python nn_entropy_simulation.py
```

**输出**：
- `nn_entropy_simulation.png`：训练准确率与权重熵变化曲线
- 控制台输出相关系数

**验证预言**：
准确率提升期间，权重熵减少，相关系数>0，符合论文7.1节预言 η_intelligence ∝ I_mutual。

## 📊 结果解读

### 时间场模拟结果
- 低熵区域（智能系统）对应时间场负值（时间-相态）
- 熵密度高的区域时间场演化更快
- 符合时间-熵耦合方程：dS/dτ = α d<T>/dt

### 神经网络模拟结果
- 训练过程中，准确率持续上升，权重熵持续下降
- 准确率提升速率与熵减速率正相关
- 符合智能熵减假说：智能系统通过信息处理实现局部熵减
